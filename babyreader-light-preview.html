让文字回到白纸上

BabyReader Light for Typora 是深色 BabyReader 的浅色 companion。它不是纯白默认主题，而是借用右图那种公众号预览的白纸、暖金边线和浅米色块，保留 BabyReader 的安静版心。

正文使用深暖灰，链接使用 暖金色强调，行内代码像 #c4a06a 一样轻轻浮出来，不打断阅读。

如果深色版像夜间阅读器，浅色 companion 更像一张被整理好的白纸。它应该清楚，但不能刺眼。

是 companion，不是反相

浅色版沿用 720px 版心、宋体标题、1.9 行高、克制链接和代码块。变化主要在色彩层级：背景更接近米白，正文落在暖灰，强调色从陶土橙转成更适合白底的暖金。

适合的 Markdown 元素

  • 引用块使用浅米底和暖金边线，参考附图右侧的预览框。
  • 二级标题保留细分割线，形成文章结构。
  • 代码块保持整体盒子，不再出现一行一个框。
:root {
  --br-bg: #fbfaf7;
  --br-paper: #ffffff;
  --br-accent: #c4a06a;
}
层级 颜色 用途
白纸 #ffffff 正文阅读区域
暖金 #c4a06a 链接、边线、分割线
浅米 #fcf8f2 引用与低调块面
